Overview

SwiftRef APIs provide financial institutions, corporates, and third-party service providers with automated data lookup services for real-time identification and validation of payments reference data.

Getting Started & Sandbox

  1. Create an app and select SwiftRef API.
  2. Obtain sandbox credentials (for password grant type as applicable).
  3. Use the Postman collection provided to test sample API calls.

Note: The sandbox environment behaves identically to live for testing. Be mindful of quota limits.
